Adobe DreamWeaver

Explique o que é e o que podemos fazer com o DreamWeaver

É um software de desenvolvimento web que permite criar, editar e gerenciar sites e aplicações web. Ele combina com uma interface visual com editor de códigos

Podemos criar sites, páginas da Internet, montar lojas virtuais, plataformas de notícias e outros diversos tipos de conteúdos digitais para colocar na Internet.

Como trabalhamos ou fazemos.

Temos duas opções de usarmos o DreamWeaver, podemos criar os conteúdos dos sites incorporando os elementos por itens nos Menus (Visual), ou criarmos o site programando (TAGs) com códigos em HTML, CSS e outras linguagens de programação Web.

D- Explique o HTML 

) Permitem que você mova os documentos para frente e para trás entre um editor HTML baseado em texto e o Dreamweaver com pouco ou nenhum efeito no conteúdo e na estrutura do código-fonte HTML original do documento. 

E- O que são as TAGs, descreva exemplos

) As tags são etiquetas que não aparecem na tela do computador, mas que servem para orientar alguém. Os robot que são os programas dos mecanismos de busca e que vasculham as páginas da web e recolhe as informações contidas nessas etiquetas.

F- Explique o CSS.

É um código em que você pode fazer alterações rápidas de layout, como definição de cores e fontes, por exemplo.
